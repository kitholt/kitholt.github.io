临时抱佛脚之数据库

面向死记硬背的Oracle数据库

因为懒得打开学习通

第一单元：实例

  1. Oracle进程结构中完成更新控制文件与数据文件,使其同步并触发DBWR进程,使其将脏缓存块写入数据文件中的进程是( )。—后台进程CKPT
  2. 当Oracle服务器启动时,下列哪种文件不是必须的( )。—归档日志文件
  3. ( )是数据库数据字典的所有者。—sys
  4. 在同一个表空间中，可以有多个但是不会存在重复的是（）—文件号
  5. 可以在Oracle 11g服务器的安装目录下的( )文件中查看Oracle11g服务器的各种参数的使用情况。—initorcl.ora
  6. Oracle数据库的实例由( )组成。—SGA(系统全局区)和后台进程
  7. 以下内存区不属于实例的是( )。—PGA
  8. 如果服务器进程无法在数据缓冲区中找到空闲块,为添加从数据文件中读出的数据块,则系统会启动如下哪一个进程( )。—DBWR
  9. 数据字典表和视图存储在( )。—SYSTEM表空间
  10. Bob试图正常关闭数据库,Oracle显示实例处于空闲状态,他试图启动数据库,Oracle显示数据库已启动,Bob最好使用( )命令关闭数据库。—ABORT
  11. Oracle进程结构中完成将重做记录从重做日志高速缓冲区写入重做日志文件的进程是( )。—后台进程LGWR
  12. 将修改过的数据块写入数据文件的进程是( )。—DBWR
  13. 关闭数据库的各种方式中,最省时又不会出现数据不一致(不一致即需要恢复数据库)的情况的是( )。—immediate
  14. 使用哪一条SHUTDOWN语句关闭数据库时,下次打开数据库是需要做恢复操作( )。—
    SHUTDOWN ABORT
  15. 指定( )表空间参与排序操作,可以提高排序效率。—TEMPORARY
  16. 最近使用过的SQL语句存储在( )。—共享池
  17. 下列哪种表空间可以被设置为脱机状态( )。—用户表空间
  18. ( )不属于数据库物理存储结构。—表空间
  19. Oracle分配磁盘空间的最小单位是( )。—数据块
  20. Oracle 数据库启动时需要经历三个步骤和状态变换,以下顺序正确的是( )。—nomount→mount→open
  21. 回退段存储( )。—事务修改的数据原值
  22. 段是表空间中一种逻辑存储结构,以下( )不是ORACLE数据库使用的段类型。—代码段.
  23. 回退段的作用是( )。—数据库恢复、事务回滚、数据的读一致性
  24. 已提交的事务所做的数据改变由( )进程记录。–-DBWR
  25. Oracle中标准的数据块大小应该由参数 ( )决定。—DB_BLOCK_SIZE
  26. Oracle事务提交即( ) 。—LGWR进程成功写入日志
  27. 数据块的哪三个部分合称为块头部?( )—表目录、行目录、标题
  28. 数据库启动过程中哪一步读取初始化参数文件?( )—实例启动
  29. ( )是用于保存数据库的所有变化信息的,从而保护数据库的安全。—重做日志文件
  30. ( )是较小的二进制文件,用来描述数据库结构。—控制文件
  31. 启动DBWR进程的条件有( )。—当数据缓存的LRU列表的长度达到初始化 DB_ BLOCK _WRITE _BATCH指定值的一半时、当检查点发生时,将启动DBWR进程、若发生超时(大约3秒未被启动)、执行INSERT、UPDATE等操作时 ,没有足够的空闲块
  32. 下面属于SGA组成部分的是( )。—数据高速缓冲区、大型池、共享池、Java池
  33. 启动LGWR进程的条件包括( )。—每隔3秒,即发生一次超时,将启动LGWR、重做日志缓存被写满三分之一、用户通过COMMIT语句提交当前事务、DBWR进程开始将脏缓存块写入数据文件时先启动LGWR
  34. CKTP进程一定会引起更新的文件有( )。—控制文件、联机重做日志文件、数据文件

第二单元：结构管理

  1. 在非归档日志方式下,操作的数据库禁用了( )。—归档日志

  2. 当复用了控制文件后,其中一个控制文件损坏,理想的解决方法是( )。—用好的控制文件,覆盖坏的控制文件

  3. 创建一个数据库需要多少个控制文件?( )— 1

  4. 利用哪个参数可以设置撤销空间的管理方式?( )—UNDO_MANAGEMENT

  5. Oracle数据文件的管理策略不包含( )。—重做日志文件最好与数据文件存放在同一个磁盘上。

  6. 如何修改数据库所允许的重做日志个数( )。—修改控制文件

  7. UNDO 数据的自动管理方式下( )。—使用 UNDO 表空间统一存储还原数据

  8. 控制文件中不包含( )。—实例信息

  9. 以下哪一个文件记录了在数据库恢复期间使用的检查点信息?( )—控制文件3

  10. 启动数据库时,如果一个或多个CONTROL_FILES参数指定的文件不存在或不可用,会出现什么样的结果?( )—Oracle返回警告信息,但不加载数据库

  11. 尽管oracle默认每个重做日志组中至少有一个成员,但是oracle建议在生产数据库中重做日志文件应该至少有几个成员?( )—4

  12. 增加控制文件的镜像文件,说法正确的是( )。—需要修改初始化参数文件

  13. 当创建一个新数据库时,以下哪种方法可以正确地多重映像控制文件?( )—在发CREATE DATABASE语句之前使用参数文件中的CONTROL_FILES初始化参数来说明至少两个控制文件的名字和位置

  14. 关于脱机表空间的说法哪一个是正确的?( )—如果将表空间设置为脱机状态,下次启动数据库时,不会对该表空间的数据文件进行可用性检查。

  15. 对撤销表空间不能执行哪些操作?( )—CREATE TABLE

  16. 在创建数据库时,必需的表空间是( )。—SYSTEM TABLESPACE

  17. DBA使用( )命令显示当前归档状态。—ARCHIVE LOG LIST

  18. DBA 在执行下列语句时返回了错误,DROP TABLESPACE USRES;可能的原因是下列哪个?( )—该表空间为非空,包含数据对象。

  19. 如果UNDO空间有活动事务,那么( )。—可以正常切换,切换后活动事务对应的撤销段保留在旧的 UNDO 表空间。

  20. 如果日志文件尺寸过小,最可能引起的问题是?( )—日志经常处于等待

  21. 在为已有的数据库增加重做日志文件时,数据库应处于哪种状态?( )—OPEN

  22. 当创建控制文件时,数据库必须处于什么状态?( )—未加载

  23. 在( )状态下可对重做日志组进行删除。—inactive

  24. 哪一个表空间不能切换为脱机状态?( )—系统表空间

  25. 假如一个表空间只具有一个大小为 100MB 的数据文件,现需将该数据文件的大小
    修改为 10MB,哪种方法是正确的?( )—使用带有 RESIZE 子句的 ALTER DATABASE DATAFILE 语句

  26. 如果某个数据库拥有两个重做日志组,但是第 2 重做日志组突然损坏(其中所有的
    成员日志都不可用),这时 DBA 应采取下列哪项操作?( )—删除原有的第 2 重做日志组,然后再使用 ALTER DATABASE ADD LOGFILE GROUP2 语句建立新的第 2 重做日志组。

  27. 表空间与数据文件关系属性正确的是( )。—一个数据文件只可属于一个表空间

  28. 把多路复用控制文件存储于不同磁盘的最大好处是( )。—防止失败

  29. 以下哪个不是撤销段的作用( )。—存储数据库对象

  30. 一个实例最多可以建立( )个归档进程。—10

  31. 删除表空间数据文件的说明正确的是( )。—只能删除空的数据文件。

  32. 在向一个现有的数据库中添加了一个数据文件之后,紧接着要做的操作是什么?( )—*备份控制文件

  33. 启动Oracle Database 11g至少需要( )个联机重做日志组。—2

  34. 如果某个数据库 LGWR 进程经常会因为检查点未完成而进入等待状态,DBA 应当采取什么措施来解决这个问题?( )—增加新的重做日志组

  35. 如果需要将oracle数据库的模式从NOARCHIVE 改为ARCHIVE模式,需要在什么状态进行?( )—mount

  36. 用于显示所有表空间描述信息的视图是（ ）—v$tablespaces

  37. 表空间管理策略,正确的是( )。—A、

    [将表空间的数据文件分散保存到不同的硬盘上,平均分布物理I/O操作。]

    B、

    [将回滚数据与用户数据分离,避免硬盘损坏而导致永久性的数据丢失。]

    C、

    [将数据字典与用户数据分离,避免由于数据字典对象和用户对象保存在一个数据文件中而产生I/O冲突。]

    D、

    [能够将表空间设置为脱机状态或联机状态,以便对数据库的一部分进行备份或恢复。]

    E、

    [为不同的应用创建独立的表空间,避免多个应用之间的相互干扰。]

    F、

    [能够将表空间设置为只读状态,从而将数据库的一部分设置为只读状态。]

  38. 关于联机重做日志,以下哪两个说法是正确的?( )—同一组中的所有成员文件都是同样大小、同一组的成员文件应置于不同的磁盘、同一组的成员文件内容完全相同

  39. 在以下有关控制文件设置与备份的论述中,哪个是正确的?( )—应该在向一个现有的数据库中添加一个数据文件之后备份控制文件、应该在创建一个表空间之后备份控制文件

第三单元：模式对象

  1. 在Oracle中创建用户时,比较适合DEFAULT TABLESPACE的表空间是( )—USERS
  2. 关于创建表时,对于表的存储参数设置说法错误的是()—同一个表空间中的表的存储参数不能冲突
  3. 关于索引使用的说法,错误的是()—Oracle对索引使用的决定都是最优的
  4. 关于模式的描述下列哪一项不正确( )。—一个表可以属于多个模式
  5. 下列的约束中,不会自动创建索引的是()—foreign key
  6. 什么是模式?( )—数据库对象的逻辑组织结构
  7. 在Oracle中,一个用户拥有的所有数据库对象统称为 ( )。—模式
  8. 下列( )不是Oracle系统中提供的约束。—控制约束
  9. 在Oracle中当一个数据表产生了大量存储碎片时,最好的调整方式是()—表结构重组
  10. 关于索引的存储,说法错误的是()—索引的存储参数必须跟表的参数一致
  11. 表结构重组时,不会发生变化的是()—段的名称
  12. 关于约束的延迟检查,说法错误的是()—可以一次性将某个表中所有的可延迟的约束设置为延迟
  13. 分区表中的索引类型，不存在的是（）—本地非分区索引
  14. 下列不属于分区表类型的是（）—Time分区
  15. 唯一值约束和主键约束的区别是什么?( )—唯一值约束的列可以为空、主码列不可以为空
  16. 关于约束禁用与激活,说法错误的是()—禁用主键约束、惟一性约束时,会删除其对应的索引,无法保留。
  17. 删除表时,不会删除的内容是()—与该表相关的区
  18. 关于B树索引的效率与数据量相关性低的原因,正确的是()—B树索引会优先扩展同层节点,而不是增加层数
  19. 关于分区表中各个分区的数据存储，说法正确的是（）—可以忽略分区，直接向表中添加数据

第四单元：安全与备份

  1. 关于闪回技术,说法错误的是( )—闪回技术可以保护系统表空间中的表。
  2. DML误操作可以使用( )来恢复。—还原段
  3. 创建用户的命令中,( )子句限制用户对象在数据库中占用的空间。—QUOTA
  4. 数据库运行在归档模式时,关于RMAN操作的描述正确的是( )。—RMAN需要恢复目录
  5. 完全备份时,数据库应该处于( )—mount状态
  6. 下面不是常用的对象权限的是( )。—grant
  7. 要恢复一个丢失的数据文件,而归档重做日志文件已经损坏,DBA应该执行的命令是( )。—RECOVER DATABASE UNTIL CANCEL;
  8. 使用RMAN进行数据库不完全恢复,目标数据库必须处于的状态为( )。—MOUNT
  9. 下列关于角色描述,说法有误的是( )—角色可以授予自己,也可以循环授予。
  10. 以下哪个权限是对象权限?( )—INDEX
  11. Oracle有不同类型的备份,不包括( )。—人工备份
  12. 利用下面闪回技术不能直接进行数据恢复的是( )。—闪回事务查询
  13. 为了执行一次完整的数据库介质恢复操作,数据库必须处于那种状态?( )—处于Mount状态,但不打开数据库
  14. 当数据库系统崩溃以后首要任务是()。—让数据库开启并对外提供服务
  15. 数据库运行在非归档模式,数据库的一个表空间不可用,DBA恢复数据库的方法是( )。—利用数据库的一致性备份修复数据文件、控制文件和重做日志文件
  16. 修复Recover是利用重做日志文件或数据库的增量备份文件重新建立已经丢失的数据,这是一种( )。—逻辑修复
  17. 概要文件中SESSIONS_PER_USER限制了什么?( )—每用户会话数量
  18. 若用户要连接数据库,则该用户必须拥有的权限是( )。—create session
  19. 哪个参数限制用户在断开前的最大空闲时间?( )—IDLE_TIME
  20. 给用户定义缺省角色时,下列哪种情况不行? ( )—当角色有密码时
  21. 关于闪回技术的描述正确的是( )。—闪回技术是数据库物理备份与恢复、逻辑备份与恢复的重要补充
  22. 使用( )命令,被授予权限的用户可以进一步将这个权限授予其他用户。—WITH ADMIN OPTION
  23. 以下权限哪个是系统权限?( )—BACKUP ANY TABLE
  24. 概要文件不能限制下列哪个资源的使用?( )—读取数据块时间
  25. 数据库处于归档模式,不可进行( )类型的备份。—使用ALTER DATABASE BACKUP TABLESPACE备份联机表空间
  26. 以下4种关闭数据库的方式中,只有( )在系统重新启动时需要进行数据库的恢复。—shutdown abort
  27. 以下哪种备份方式需要完全关闭数据库( )—非归档日志模式下的数据库备份
  28. 在数据库打开状态下进行备份时,下面描述不正确的是( )。—数据库必须处于READ ONLY状态
  29. Oracle数据库利用( )进行资源限制和口令管理。—概要文件
  30. 在Oracle数据库中,管理员以管理员身份登录后,进入的模式是( )。—SYS
  31. 闪回数据库功能需要满足的条件有( )。—数据库必须处于归档模式、数据库设置了闪回恢复区、数据库必须使用管理员帐号登录、数据库启用了FLASHBACK DATABASE特性
  32. 用户属性包括( )。—A、账户状态B、表空间配额C、临时表空间D、默认表空间E、概要文件F、用户身份认证方式
  33. 概要文件资源限制类型包括( )。—用户连接数据库的总时间、CPU使用时间、用户连接数据库的空闲时间、逻辑读时间
  34. 备份原则与策略正确的有( )。—A、对于重要的表中的数据,可以采用逻辑备份方式进行备份。 B、在归档模式下,当数据库结构发生变化时,如创建或删除表空间、添加数据文件、重做日志文件等,应该备份数据库的控制文件 C、在刚建立数据库时,应该立即进行数据库的完全备份 D、至少保证两个归档重做日志文件的归档目标,不同归档目标应该分散于不同磁盘 E、将所有的数据库备份保存在一个独立磁盘上 F、根据数据库数据变化的频率情况确定数据库备份规律
